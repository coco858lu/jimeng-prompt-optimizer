API 配置

选择 AI 模型 provider 并配置 API Key

豆包 (火山引擎 Ark)

在火山引擎控制台 -> Ark 推理获取 API Key

支持多模态的模型才能使用参考图功能

DeepSeek

从 platform.deepseek.com 获取 API Key

DeepSeek 目前不支持多模态，无法使用参考图功能

通义千问 (阿里云百炼)

从 阿里云百炼控制台 获取 API Key (DASHSCOPE_API_KEY)

多模态模型: qwen-vl-max (支持参考图)。纯文本模型: qwen-max

自定义 OpenAI 兼容 API

💡 支持的格式: OpenAI 兼容 API，必须支持 /chat/completions 接口。

如需多模态支持（参考图功能），请使用支持图片输入的模型（如 GPT-4o）。

优化模板

自定义提示词优化模板。使用 {prompt} 占位符表示原始提示词位置。

📖 使用说明

  1. 在上方填写对应 AI 模型的 API Key
  2. 点击「保存设置」
  3. 打开 即梦网页版
  4. 点击浏览器工具栏的扩展图标打开优化器
  5. 点击「读取」获取当前输入的提示词（或手动输入）
  6. 选择优化模板，点击「开始优化」
  7. 优化完成后可「复制」或「回填」到即梦页面
